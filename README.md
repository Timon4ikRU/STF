# STF (Shortened Text Format)

A custom binary file format with built-in compression for efficient text storage.

## Features

- **Custom 8-bit encoding** - Supports Russian/English letters, digits, symbols
- **Dual formats** - STF (fast) and ZSTF (compressed)
- **Metadata** - Built-in creation timestamp
- **Cross-platform** - Pure Python implementation
- **CLI tools** - Easy conversion and viewing

## Installation

git clone https://github.com/Timon4ikRU/STF.git
